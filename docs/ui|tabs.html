ui/tabs

inherits: can.Control

A Tabs widget for showing and hiding content.

new Tabs(element, options)

Parameters

  1. element {jQuery | HTMLElement | String}

    An element the tabs will be created on.

  2. options {Object}

    A object of option name-value pairs.

    • enabled {can.compute}

      Enabled is a can.compute that specifies which tabs should be enabled. The compute should return an array of tab content ids.

Use

Given content like:

<ul id='resultsTab'>
  <li><a href='#flickr'>Flickr</a></li>
  <li><a href='#yahoo'>Yahoo</a></li>
  <li><a href='#upcoming'>Upcoming</a></li>
</ul>

<div id='flickr'></div>
<div id='yahoo'></div>
<div id='upcoming'></div>

Create a Tabs like:

new Tabs("#resultsTab",{
  enabled: can.compute(["flickr","yahoo"])
})

Notice that each li should have an href that points to an id of the tab content to be shown.
