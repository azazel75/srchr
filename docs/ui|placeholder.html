ui/placeholder

inherits: can.Control

new Placeholder(element, options) creates a placeholder effect for browsers that do not support it.

new srchr/placeholder(element, options)

new Placeholder(element, options) creates a placeholder effect for browsers that do not support it.

Parameters

  1. element {HTMLInputElement}

    the element to show results within.

  2. options {Object}

    An object of the following options:

    placeholder can.Compute or String

    The placeholder text of the element.

new Placeholder('#search',{
 placeholder: "Enter search text"
});
