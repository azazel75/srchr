Srchr

Srchr is a crowdsourced exercise in collecting JavaScript wisdom. For more information about the project, visit http://blog.rebeccamurphey.com/2010/03/15/srchr-crowdsourcing-javascript-wisdom/.

This is a JavaScriptMVC version created by Bitovi.

Installing

Clone srchr and install its submodules:

git clone git://github.com/bitovi/srchr srchr
cd srchr
git submodule update --init --recursive

Running

Open srchr/srchr.html.

Testing

In your browser, open [//srchr/test.html]. To run it automated, open an console to the srchr folder and run:

./js funcunit/open/selenium http://localhost/srchr/srchr/test.html

Note: Replace http://localhost/srchr with the path to srchr.

Building

Open a console to the srchr folder and run:

./js srchr/scripts/build.js

Open srchr/srchr.html in a text editor and change:

<script src='../steal/steal.js?srchr'>

to

<script src='../steal/steal.production.js?srchr'>

Open srchr/srchr.html in your browser.
